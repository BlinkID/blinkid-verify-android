VerifySdkStrings

@Immutable
data class VerifySdkStrings(val scanningStrings: ScanningStrings, val helpDialogsStrings: HelpDialogsStrings)

Data class contains all the strings used throughout the SDK. Default can be used to keep the original strings if only some of the elements are to be changed.

This class shouldn't be modified, but rather a new instance should be created and used in com.microblink.blinkidverify.ux.VerifyUiSettings.verifySdkStrings when creating an instance of com.microblink.blinkidverify.ux.CameraScanningScreen.
