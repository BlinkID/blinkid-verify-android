ProcessingState

enum ProcessingState : Enum<ProcessingState>

Represents the different states of the reticle and UI elements during the scanning process.

This enum class defines the various states that the UI can be in while a document is being captured and processed. Each state is associated with a specific ReticleState and a minimum duration.
